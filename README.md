​ نبذة عن المشروع
​"مَيْسّر" هي منصة استشارية ذكية تهدف إلى مساعدة الأفراد في معالجة التعقيدات المالية والإدارية والقانونية البسيطة.

 تعتمد المنصة على محاكاة نظام وكلاء متعددين (Multi-Agent System) لتحليل الحالات، تقديم التوصيات المالية، وصياغة الخطابات الرسمية الموجهة للجهات المختصة بدقة وسرعة.

​🎯 المشكلة التي يحلها المشروع
​الكثير من الأفراد يواجهون صعوبة في صياغة الخطابات الرسمية أو فهم كيفية إعادة هيكلة التزاماتهم المالية عند مواجهة أزمات طارئة (مثل المخالفات أو العجز المالي المؤقت)، مما يتطلب حلولاً سريعة ومنظمة توفر الوقت والجهد.

​⚙️ آلية عمل النظام (الوكلاء المتعددون)
​تقوم المنصة بمعالجة المدخلات من خلال 3 مراحل متسلسلة عبر وكلاء أذكياء:

​وكيل التصنيف (Classifier Agent): تحليل وتصنيف بنية البيانات واستخراج المتغيرات الحرجة.

​الوكيل المالي (Financial Engine): حساب التدفقات النقدية، مؤشرات العجز، وتقديم خطط إعادة الهيكلة.

​الوكيل القانوني (Legal Drafter): صياغة الخطابات الرسمية الموجهة للجهات المختصة بأسلوب نظامي ومؤثر.
